# deep learning

## ch01 パーセプトロン

線形代数

- $y = w x + b$
- $y$ : 出力　$w$: 重み　$x$: 入力　$b$: バイアス
- 目的: 0, 1 のコンピュータを作成したい

- 多次元データの場合
- $\bold{y} = \bold{w} \bold{x}+ \bold{b}$ で表せる

## ch02 論理回路

ゲートの種類

- AND
- OR
- NAND
- XOR

- 自動でパラメータを調整したい
- $\bold{y} = h(\bold{x})(\bold{w} \bold{x/b} + \bold{b})$
- h(\bold{x})を活性化関数と呼ぶ
